# 🌍 AI Smart Campus Digital Twin System

An AI-powered digital twin system designed to monitor, analyze, and optimize sustainability in campus environments.

---

## 🚀 Project Overview

This project simulates a **smart campus environment** using a digital twin approach.  
It helps track and improve key sustainability factors such as:

- ⚡ Energy Consumption  
- 🍽 Food Waste  
- 💧 Water Usage  

The system provides **real-time insights, predictions, and intelligent recommendations** to support better decision-making.

---

## 🎯 Key Features

- 📥 **Data Input Module**  
  Enter real-world campus data for analysis  

- 📊 **Interactive Dashboard**  
  Visualize trends using graphs and metrics  

- 🔮 **AI-Based Prediction**  
  Predict future resource usage using Machine Learning  

- 🚨 **Smart Insights & Alerts**  
  Detect inefficiencies automatically  

- 🤖 **Smart Recommendations**  
  Suggest ways to reduce energy, waste, and water usage  

- 🧠 **What-if Simulation**  
  Test different scenarios and observe sustainability score changes  

---

## 🛠️ Technologies Used

- 🐍 Python  
- ⚡ Streamlit  
- 📊 Pandas  
- 🤖 Machine Learning (Linear Regression)  

---

## 🧪 How It Works

1. User inputs sustainability data  
2. System analyzes and visualizes the data  
3. Machine learning predicts future trends  
4. AI generates insights and recommendations  
5. Simulation allows testing of different scenarios
